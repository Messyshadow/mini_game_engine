# miniaudio

本目录需要放置 `miniaudio.h` 文件。

## 下载方法

### 方法1：运行脚本
```
运行项目根目录的 download_miniaudio.bat
```

### 方法2：手动下载
1. 访问: https://raw.githubusercontent.com/mackron/miniaudio/master/miniaudio.h
2. 保存为: depends/miniaudio/miniaudio.h

### 方法3：使用curl
```bash
curl -L -o depends/miniaudio/miniaudio.h https://raw.githubusercontent.com/mackron/miniaudio/master/miniaudio.h
```

## 关于miniaudio

- 官网: https://miniaud.io/
- GitHub: https://github.com/mackron/miniaudio
- 许可证: Public Domain / MIT
- 特点: 单头文件、跨平台、支持多种格式(WAV, MP3, FLAC等)
