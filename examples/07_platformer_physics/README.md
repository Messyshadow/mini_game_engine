# 第七章：平台物理系统

## 📚 本章概述

本章将实现2D平台跳跃游戏的核心物理系统，这是《空洞骑士》《蔚蓝》《银河战士》等游戏的基础。

### 学习目标

1. 理解重力和速度的物理模拟
2. 实现可变高度的跳跃
3. 处理地面、墙壁、天花板碰撞
4. 实现单向平台（可从下方穿过）
5. 理解物理更新的时间步进

## 🎓 理论知识

### 7.1 基础物理概念

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           物理模拟基础                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  在游戏物理中，我们模拟的是牛顿运动定律的简化版本：                            │
│                                                                             │
│  位置(Position)、速度(Velocity)、加速度(Acceleration) 的关系：              │
│                                                                             │
│      加速度 → 改变速度 → 改变位置                                           │
│                                                                             │
│  每帧更新：                                                                  │
│      velocity += acceleration * deltaTime                                  │
│      position += velocity * deltaTime                                      │
│                                                                             │
│  示例（自由落体）：                                                          │
│                                                                             │
│      gravity = -980.0  (向下的加速度，单位：像素/秒²)                        │
│                                                                             │
│      帧1: velocity.y = 0 + (-980) * 0.016 = -15.68                         │
│           position.y = 100 + (-15.68) * 0.016 = 99.75                      │
│                                                                             │
│      帧2: velocity.y = -15.68 + (-980) * 0.016 = -31.36                    │
│           position.y = 99.75 + (-31.36) * 0.016 = 99.25                    │
│                                                                             │
│      ... 速度越来越快，这就是加速下落！                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 7.2 跳跃的实现

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           跳跃的物理原理                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  跳跃 = 给予一个向上的初速度，然后让重力自然作用                              │
│                                                                             │
│  velocity.y                                                                │
│      ▲                                                                     │
│  400 │    ●  ← 跳跃瞬间，给予向上初速度                                    │
│      │     ╲                                                               │
│  200 │      ╲                                                              │
│      │       ╲                                                             │
│    0 │────────●────────────→ 时间                                          │
│      │         ╲    ↑ 到达最高点，速度为0                                  │
│ -200 │          ╲                                                          │
│      │           ╲                                                         │
│ -400 │            ● ← 开始下落，速度变负                                   │
│      │                                                                     │
│                                                                             │
│  代码实现：                                                                  │
│                                                                             │
│      if (按下跳跃键 && 在地面上) {                                          │
│          velocity.y = jumpForce;  // 例如 400                              │
│          onGround = false;                                                 │
│      }                                                                      │
│                                                                             │
│      // 每帧应用重力                                                        │
│      velocity.y += gravity * deltaTime;  // gravity 是负值                 │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 7.3 可变高度跳跃

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        可变高度跳跃（Variable Jump）                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  优秀的平台游戏都支持"轻按小跳，长按大跳"：                                   │
│                                                                             │
│      长按跳跃键              轻按跳跃键                                      │
│                                                                             │
│          ╱╲                      ╱╲                                        │
│         ╱  ╲                    ╱  ╲                                       │
│        ╱    ╲                  │    │                                      │
│       ╱      ╲                 │    │                                      │
│      ╱        ╲                ╯    ╰                                      │
│     ●──────────●              ●──────●                                     │
│     高跳跃                     低跳跃                                       │
│                                                                             │
│  实现原理：                                                                  │
│                                                                             │
│  方法1：松开按键时减速                                                       │
│      if (松开跳跃键 && velocity.y > 0) {                                   │
│          velocity.y *= 0.5;  // 立即减半上升速度                            │
│      }                                                                      │
│                                                                             │
│  方法2：不同的重力                                                           │
│      if (velocity.y > 0 && !按住跳跃键) {                                  │
│          // 上升时松开按键，使用更大的重力                                   │
│          velocity.y += gravity * 2.5 * deltaTime;                          │
│      } else {                                                               │
│          velocity.y += gravity * deltaTime;                                │
│      }                                                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 7.4 碰撞检测与响应

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           碰撞检测流程                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  正确的碰撞检测需要分轴处理：                                                 │
│                                                                             │
│  错误方式（同时移动X和Y）：                                                   │
│                                                                             │
│      position += velocity * deltaTime;  // 可能穿墙！                       │
│                                                                             │
│          ┌─────┐                                                           │
│          │     │                                                           │
│      ●───┼─────┼───●   玩家从左边穿到右边                                  │
│          │     │                                                           │
│          └─────┘                                                           │
│            墙壁                                                             │
│                                                                             │
│  正确方式（分轴移动）：                                                       │
│                                                                             │
│      // 步骤1：先移动X轴                                                    │
│      position.x += velocity.x * deltaTime;                                 │
│      处理X轴碰撞();  // 如果撞墙，推出来                                    │
│                                                                             │
│      // 步骤2：再移动Y轴                                                    │
│      position.y += velocity.y * deltaTime;                                 │
│      处理Y轴碰撞();  // 如果撞地面/天花板，推出来                            │
│                                                                             │
│          ┌─────┐                                                           │
│          │     │                                                           │
│      ●───●     │       玩家被正确阻挡在墙外                                 │
│          │     │                                                           │
│          └─────┘                                                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│           │     │                                                          │
│           └─────┘                                                          │
│           ┌─────┐  ← 脚底检测盒（很薄，比如2像素高）                        │
│           └─────┘                                                          │
│      ════════════════  地面                                                │
│                                                                             │
│      onGround = 脚底检测盒与地面重叠;                                       │

```

### 7.6 单向平台

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        单向平台（One-Way Platform）                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  单向平台：可以从下方跳上去，但不能从上方穿过                                  │
│                                                                             │
│      从下往上跳：                    从上往下：                              │
│                                                                             │
│          ▲                           ●                                     │
│          │                           │                                     │
│          ●                           ▼                                     │
│      ════════════              ════════════                                │
│       可以穿过！                 站在上面！                                  │
│                                      ●                                     │
│                                                                             │
│  实现原理：                                                                  │
│                                                                             │
│      只有当以下条件都满足时才处理碰撞：                                       │
│      1. 玩家正在下落（velocity.y < 0）                                      │
│      2. 玩家脚底在平台顶部上方（player.bottom >= platform.top）             │
│                                                                             │
│      伪代码：                                                                │
│      if (platform.isOneWay) {                                              │
│          // 只有从上方落下时才碰撞                                          │
│          if (velocity.y >= 0) continue;  // 上升时忽略                     │
│          if (player.bottom < platform.top - 1) continue;  // 在下方忽略    │
│      }                                                                      │
│      // 处理碰撞...                                                         │
│                                                                             │
│  下跳穿过单向平台：                                                          │
│                                                                             │
│      if (按下↓ && 按下跳跃 && 站在单向平台上) {                             │
│          // 暂时禁用该平台的碰撞                                            │
│          position.y -= 2;  // 向下移动一点，脱离平台                        │
│      }                                                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 💻 代码实现

### 核心类：PhysicsBody2D

```cpp
/**
 * PhysicsBody2D - 2D物理体
 * 
 * 属性：
 * - position: 位置
 * - velocity: 速度
 * - acceleration: 加速度（通常只有重力）
 * - gravityScale: 重力缩放（0=不受重力，1=正常，2=双倍）
 * 
 * 每帧更新：
 *   velocity += gravity * gravityScale * deltaTime
 *   position += velocity * deltaTime
 */
```

### 核心类：PlatformPhysics

```cpp
/**
 * PlatformPhysics - 平台物理系统
 * 
 * 职责：
 * 1. 管理所有碰撞体（地面、平台、墙壁）
 * 2. 处理角色与地形的碰撞
 * 3. 提供地面检测、墙壁检测等功能
 * 
 * 主要方法：
 * - MoveAndCollide(): 移动并处理碰撞
 * - IsOnGround(): 检测是否在地面上
 * - IsOnWall(): 检测是否贴墙
 */
```

|------|------|

## 🔧 参数调节指南

### 跳跃手感调节

```cpp
// 基础参数
float gravity = -1500.0f;       // 重力加速度
float jumpForce = 500.0f;       // 跳跃初速度
float moveSpeed = 300.0f;       // 移动速度

// 高级参数
float fallMultiplier = 2.5f;    // 下落时的重力倍数（让下落更快）
float lowJumpMultiplier = 2.0f; // 轻跳时的重力倍数

// 计算跳跃高度（物理公式）
// h = v² / (2 * g)
// 例如：500² / (2 * 1500) = 83.3 像素

// 计算跳跃时间（到达最高点）
// t = v / g
// 例如：500 / 1500 = 0.33 秒
```

### 不同游戏风格的参数参考

| 游戏风格 | 重力 | 跳跃力 | 特点 |
|----------|------|--------|------|
| 马里奥风格 | -1200 | 450 | 飘逸、滞空长 |
| 银河战士风格 | -1800 | 550 | 沉重、精准 |
| 蔚蓝风格 | -1500 | 500 | 平衡、可控 |
| 空洞骑士风格 | -2000 | 600 | 快速、紧凑 |
