# 第18章：完整银河恶魔城Demo

## 概述

2D项目最终总结篇。整合前17章所有技术，新增武器系统、技能系统、瞬移残影、编辑模式增强，做出一个完整可玩的银河恶魔城风格游戏。

## 游戏流程

```
标题画面 → 大厅(教学+二段跳) → 走廊(战斗+剑) → 实验室(探索+锤+瞬移) → 守卫室(锁门战) → BOSS → 通关
```

## 五个关卡

| 房间 | 拾取物 | 敌人 | 特色 |
|------|--------|------|------|
| Entrance Hall | 二段跳 | 1近战 | 教学，简单平台 |
| Mechanical Corridor | 墙跳 + 长剑 | 2近战+1远程 | 战斗练习 |
| Underground Lab | 瞬移 + 重锤 | 1混合+1远程+1近战 | 竖直结构 |
| Elite Guard Room | — | 1混合+1近战+1远程 | 锁门，全灭才开门 |
| BOSS Chamber | — | BOSS(HP300)+2小兵 | 三阶段BOSS战 |

## 操作说明

### 移动

| 按键 | 功能 |
|------|------|
| A/D | 移动（有加速度曲线，非瞬间满速） |
| Space短按 | 低跳（松手削减上升速度） |
| Space长按 | 高跳（按住保持上升） |
| 空中Space | 二段跳（需拾取解锁，力度为85%） |
| 贴墙+Space | 蹬墙跳（需拾取解锁，反向弹出） |

### 战斗

| 按键 | 功能 | 消耗 |
|------|------|------|
| J | 普通攻击（当前武器） | 无 |
| L | 技能攻击（武器专属技能） | 15MP + 4秒CD |
| K | 瞬移到最近敌人身后（带残影） | 20MP + 3秒CD |
| Q | 切换已拥有的武器 | 无 |

### 系统

| 按键 | 功能 |
|------|------|
| W / ↑ | 进入传送门 |
| ESC | 暂停/继续 |
| E | 编辑模式 |
| Tab | 调试面板 |
| R | 从存档点重生 |
| Enter | 标题画面开始 / 通关返回 |

## 武器系统

### 三种武器对比

| 属性 | 拳头（默认） | 长剑（走廊拾取） | 重锤（实验室拾取） |
|------|-------------|------------------|-------------------|
| 伤害 | 12 | 18 | 30 |
| 速度 | 快（22FPS） | 中（18FPS） | 慢（12FPS） |
| 范围 | 55×45 | 85×55 | 70×65 |
| 击退 | 80 | 120 | 200 |
| 硬直 | 0.2秒 | 0.3秒 | 0.5秒 |

### 武器技能（L键）

| 武器 | 技能名 | 效果 | 消耗 |
|------|--------|------|------|
| 拳头 | 冲拳 | 短距离冲刺+攻击 | 15MP |
| 长剑 | 剑气波 | 前方300px范围AOE，蓝色光波 | 15MP |
| 重锤 | 震地 | 周围200px AOE，屏幕震动 | 15MP |

## 瞬移残影效果（K键）

按K键瞬移时：
1. 在起点位置生成4个半透明蓝色残影
2. 残影从起点到终点等距分布
3. 每个残影渐隐消失（2.5秒衰减）
4. 起点和终点都发射蓝色粒子

## 编辑模式（E键）

### Physics标签页
- 移动速度、加速度、减速度、空中加速度
- 跳跃力、短按削减比例、空中跳力
- 重力、下落加速倍率、最大下落速度
- 土狼时间开关+时长、跳跃缓冲开关+时长
- 二段跳开关、墙跳开关+墙面滑行速度+墙跳力度

### Player标签页
- HP / Max HP（滑块 + Full Heal按钮）
- MP / Max MP（滑块 + Full MP按钮）
- 攻击力、瞬移CD、瞬移MP消耗
- 技能CD、技能MP消耗
- **能力开关**：Double Jump / Wall Jump / Teleport（可直接勾选，无需拾取）
- **武器开关**：Fist / Sword / Hammer（可直接勾选获得）

### Enemies标签页
- 敌人列表，点击展开编辑
- 修改：类型(Melee/Ranged/Hybrid)、HP、攻击力、巡逻范围
- 添加/删除敌人
- Apply Changes重建关卡

### Level标签页
- 平台列表，修改位置/尺寸/瓦片类型
- 添加/删除平台
- Rebuild Level重建

## BOSS战设计

| 阶段 | HP范围 | 行为变化 |

| 1 | 100%→60% | 正常速度，普通攻击 |
