# 第八章：角色控制器

## 📚 本章概述

本章将实现专业级的2D平台游戏角色控制器。你将学到《蔚蓝》《空洞骑士》等游戏使用的操控优化技术。

### 学习目标

1. 理解"土狼时间"（Coyote Time）的原理和实现
2. 理解"跳跃缓冲"（Jump Buffer）的原理和实现
3. 实现平滑的加速/减速曲线
4. 实现空中控制
5. （可选）实现蹬墙跳

### 8.2 土狼时间（Coyote Time）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         土狼时间 (Coyote Time)                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  名字来源：                                                                  │
│  《Looney Tunes》中的Wile E. Coyote（威利狼）跑出悬崖后，                    │
│  会在空中停留一会儿才开始下落（看向镜头的经典镜头）                           │
│                                                                             │
│                    ┌─────┐                                                 │
│                    │ 🐺  │  "等等..."                                      │
│                    └─────┘                                                 │
│   ══════════                                                               │
│                 ↓ 然后才开始下落                                            │
│                                                                             │
│  原理：                                                                      │
│  离开平台后的一小段时间内（通常0.1-0.15秒），仍然允许跳跃                     │
│                                                                             │
│  时间线：                                                                    │
│                                                                             │
│   离开平台                                                                  │
│      ↓                                                                     │
│      ├─────────────────────┬──────────────────────────→ 时间               │
│      │    土狼时间窗口     │                                               │
│      │   (约0.1秒)        │                                               │
│      │   还能跳！          │   不能跳了                                     │
│      └─────────────────────┴──────────────────────────                     │
│                                                                             │
│  代码逻辑：                                                                  │
│                                                                             │
│      // 在地面时，重置土狼时间                                               │
│      if (onGround) {                                                       │
│          coyoteTimeLeft = COYOTE_TIME;  // 例如 0.1秒                      │
│      } else {                                                               │
│          coyoteTimeLeft -= deltaTime;   // 离开地面后开始倒计时             │
│      }                                                                      │
│                                                                             │
│      // 跳跃条件：在地面 或 还在土狼时间内                                   │
│      bool canJump = onGround || coyoteTimeLeft > 0;                        │
│                                                                             │
│      if (jumpPressed && canJump) {                                         │
│          velocity.y = jumpForce;                                           │
│          coyoteTimeLeft = 0;  // 用掉土狼时间                              │
│      }                                                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 8.3 跳跃缓冲（Jump Buffer）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        跳跃缓冲 (Jump Buffer)                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  问题场景：                                                                  │
│  玩家在落地前一点点按下跳跃键，结果不跳                                       │
│                                                                             │
│      ┌─────┐                                                               │
│      │玩家 │  ← 按下跳跃（还在空中，不能跳）                                │
│      └──┬──┘                                                               │
│         ↓                                                                  │
│   ══════════                                                               │
│         ●  落地（但跳跃键已经按过了）                                       │
│                                                                             │
│  解决方案：                                                                  │
│  记住跳跃输入，在落地后的短时间内自动执行                                     │
│                                                                             │
│  时间线：                                                                    │
│                                                                             │
│   按下跳跃                落地                                              │
│      ↓                    ↓                                                │
│      ├────────────────────┼────────────────────→ 时间                      │
│      │   记住这次输入     │                                                │
│      │                    │                                                │
│      │    缓冲时间窗口    │                                                │
│      │    (约0.1秒)       │                                                │
│      └────────────────────┤                                                │
│                           │                                                │
│                           ● 落地时检查：有缓冲的跳跃吗？有！执行跳跃         │
│                                                                             │
│  代码逻辑：                                                                  │
│                                                                             │
│      // 按下跳跃键时，设置缓冲时间                                           │
│      if (jumpJustPressed) {                                                │
│          jumpBufferTimeLeft = JUMP_BUFFER_TIME;  // 例如 0.1秒             │
│      }                                                                      │
│                                                                             │
│      // 缓冲时间倒计时                                                       │
│      jumpBufferTimeLeft -= deltaTime;                                      │
│                                                                             │
│      // 跳跃条件：有缓冲的跳跃 且 能跳                                       │
│      bool wantJump = jumpBufferTimeLeft > 0;                               │
│      bool canJump = onGround || coyoteTimeLeft > 0;                        │
│                                                                             │
│      if (wantJump && canJump) {                                            │
│          velocity.y = jumpForce;                                           │
│          jumpBufferTimeLeft = 0;  // 消耗缓冲                              │
│          coyoteTimeLeft = 0;      // 消耗土狼时间                          │
│      }                                                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 8.5 加速度曲线

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         移动的加速度曲线                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  没有加速度（僵硬）：                                                        │
│                                                                             │
│  速度                                                                       │
│   ▲                                                                        │
│   │     ┌──────────────────────┐                                           │
│   │     │                      │                                           │
│   │     │                      │                                           │
│   │─────┘                      └─────────→ 时间                            │
│        按下                   松开                                          │
│        立即满速               立即停止                                      │
│                                                                             │
│  有加速度（平滑）：                                                          │
│                                                                             │
│  速度                                                                       │
│   ▲         ╭──────────────────╮                                           │
│   │        ╱                    ╲                                          │
│   │       ╱                      ╲                                         │
│   │──────╱                        ╲─────→ 时间                             │
│        按下                      松开                                       │
│        渐渐加速                  渐渐减速                                   │
│                                                                             │
│  实现方法：使用 Lerp 或 MoveTowards                                         │
│                                                                             │
│      // 方法1：线性插值                                                     │
│      velocity.x = Lerp(velocity.x, targetVelocity, acceleration * dt);    │
│                                                                             │
│      // 方法2：限速趋近                                                     │
│      velocity.x = MoveTowards(velocity.x, targetVelocity, accel * dt);    │
│                                                                             │
│  参数设计：                                                                  │
│                                                                             │
│      地面加速度: 较高（响应快）                                              │
│      地面减速度: 较高（停得住）                                              │
│      空中加速度: 较低（保持惯性）                                            │
│      空中减速度: 更低（飘一点）                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 8.6 空中控制（Air Control）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           空中控制                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  物理正确 vs 游戏好玩：                                                      │
│                                                                             │
│  物理正确：跳起后完全无法改变水平方向（抛物线）                               │
│                                                                             │
│                 ●                                                          │
│                ╱ ╲                                                         │
│               ╱   ╲    ← 无法改变轨迹                                      │
│              ╱     ╲                                                       │
│             ●       ●                                                      │
│                                                                             │
│  游戏好玩：空中可以调整方向（但比地面弱）                                     │
│                                                                             │
│                 ●                                                          │
│                ╱ → ╲   ← 空中按右，轨迹右移                                │
│               ╱     ╲                                                      │
│              ╱       ╲                                                     │
│             ●         ●                                                    │
│                                                                             │
│  实现方法：                                                                  │
│                                                                             │
│      float airControl = 0.7f;  // 空中控制力（0-1）                        │
│                                                                             │
│      if (onGround) {                                                       │
│          // 地面：完全控制                                                  │
│          velocity.x = MoveTowards(velocity.x, target, groundAccel * dt);  │
│      } else {                                                               │
│          // 空中：减弱控制                                                  │
│          velocity.x = MoveTowards(velocity.x, target, airAccel * dt);     │
│          // 或者限制最大速度变化                                            │
│      }                                                                      │
│                                                                             │
│  不同游戏的空中控制风格：                                                    │
│                                                                             │
│      《马里奥》：空中控制强，可以大幅调整                                    │
│      《银河战士》：空中控制中等                                              │
│      《魂斗罗》：空中几乎无法控制                                            │
│      《蔚蓝》：空中控制强，但有最大速度限制                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 8.7 蹬墙跳（Wall Jump）- 可选

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          蹬墙跳 (Wall Jump)                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  基本流程：                                                                  │
│                                                                             │
│      1. 检测贴墙                                                            │
│      2. 贴墙时可以滑落（减速下落）                                          │
│      3. 贴墙时按跳跃 → 向反方向弹跳                                         │
│                                                                             │
│                █                                                           │
│                █                                                           │
│      ┌─────┐   █                                                           │
│      │玩家 │───█  ← 贴墙                                                   │
│      └─────┘   █                                                           │
│         ↓      █                                                           │
│      缓慢滑落  █                                                           │
│                █                                                           │
│                                                                             │
│      ┌─────┐   █                                                           │
│      │玩家 │───█  按跳跃                                                   │
│      └─────┘   █                                                           │
│          ↖     █                                                           │
│           ╲    █  向左上方弹跳                                             │
│            ●   █                                                           │
│                                                                             │
│  代码逻辑：                                                                  │
│                                                                             │
│      // 检测贴墙                                                            │
│      bool onWallLeft = physics.IsOnWall(player, true);                     │
│      bool onWallRight = physics.IsOnWall(player, false);                   │
│      bool onWall = onWallLeft || onWallRight;                              │
│                                                                             │
│      // 贴墙滑落                                                            │
│      if (onWall && !onGround && velocity.y < 0) {                          │
│          velocity.y = max(velocity.y, -wallSlideSpeed);                    │
│      }                                                                      │
│                                                                             │
│      // 蹬墙跳                                                              │
│      if (jumpPressed && onWall && !onGround) {                             │
│          float dir = onWallLeft ? 1.0f : -1.0f;  // 反方向                 │
│          velocity.x = dir * wallJumpForceX;                                │
│          velocity.y = wallJumpForceY;                                      │
│      }                                                                      │
│                                                                             │
│  高级技巧：                                                                  │
│                                                                             │
│      - 蹬墙跳后短暂禁止反向输入（防止贴回墙上）                              │
│      - 墙壁也有"土狼时间"（离开墙壁瞬间还能跳）                             │
│      - 爬墙（贴墙时按上移动）                                               │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 💻 代码实现

### 核心类：CharacterController2D

```cpp
/**
 * CharacterController2D - 2D角色控制器
 * 
 * 职责：
 * 1. 处理玩家输入
 * 2. 应用土狼时间和跳跃缓冲
 * 3. 计算加速度曲线
 * 4. 管理角色状态（地面/空中/贴墙）
 * 
 * 这个类专注于"控制手感"，
 * 物理碰撞由 PlatformPhysics 处理。
 */
```

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| A/D | 左右移动（有加速度曲线） |
| 空格 | 跳跃（支持土狼时间和跳跃缓冲） |
| 长按空格 | 高跳 |
| 贴墙+空格 | 蹬墙跳（如果启用） |
| 1 | 开关土狼时间 |
| 2 | 开关跳跃缓冲 |
| 3 | 开关加速度曲线 |
| 4 | 开关蹬墙跳 |
| R | 重置位置 |

## 🔧 参数调节指南

### 推荐初始值

```cpp
// 土狼时间和跳跃缓冲
float coyoteTime = 0.1f;        // 0.08-0.15 秒
float jumpBufferTime = 0.1f;    // 0.08-0.15 秒

// 移动
float maxSpeed = 300.0f;        // 最大速度
float groundAccel = 2000.0f;    // 地面加速度
float groundDecel = 2500.0f;    // 地面减速度
float airAccel = 1000.0f;       // 空中加速度
float airDecel = 500.0f;        // 空中减速度

// 跳跃
float jumpForce = 550.0f;       // 跳跃力
float jumpCutMultiplier = 0.5f; // 轻跳速度衰减

// 蹬墙跳（可选）
float wallSlideSpeed = 100.0f;  // 贴墙滑落速度
float wallJumpForceX = 350.0f;  // 蹬墙跳水平力
float wallJumpForceY = 450.0f;  // 蹬墙跳垂直力
```

### 不同风格参数

| 风格 | 土狼时间 | 跳跃缓冲 | 地面加速 | 空中控制 |
|------|----------|----------|----------|----------|
| 精准型 | 0.05s | 0.05s | 3000 | 低 |
| 宽容型 | 0.15s | 0.15s | 1500 | 高 |
| 《蔚蓝》风格 | 0.1s | 0.1s | 2000 | 中高 |
| 《马里奥》风格 | 0.08s | 0.1s | 1200 | 高 |
