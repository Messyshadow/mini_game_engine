# 第12章：摄像机系统

## 学习目标
1. 摄像机跟随玩家
2. 平滑移动（Lerp插值）
3. 边界限制
4. 震动效果

## 核心原理

### 坐标转换
```
屏幕坐标 = 世界坐标 - 摄像机位置
```

### 平滑跟随
```cpp
position += (target - position) * smoothSpeed * deltaTime;
```

## 操作说明

| 按键 | 功能 |
|------|------|
| A/D | 移动 |
| 空格 | 跳跃 |
| 1 | 快速跟随 (10.0) |
| 2 | 中速跟随 (5.0) |
| 3 | 慢速跟随 (2.0) |
| B | 切换边界限制 |
| Z | 摄像机震动 |
| C | 切换背景颜色 |
| R | 重置位置 |

## 地图设置

本章创建了一个80×30瓦片的大地图（2560×960像素），比屏幕（1280×720）大，需要摄像机滚动才能看到全部。包含地面、悬空平台和左右墙壁。
