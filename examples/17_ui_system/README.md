## 新增功能详解

### 1. HUD系统

屏幕左下角固定显示：

```
  ┌─ HP ████████████░░░░ (绿/黄/红色随血量变化)
  ├─ MP ██████████░░░░░░ (蓝色)
  └─ [K] 瞬移技能图标 (冷却中变灰+遮罩)
```

HUD使用独立的正交投影渲染，不跟随摄像机移动。

### 2. 伤害飘字

攻击命中时，伤害数字从被击位置向上飘动并渐隐。暴击显示为金色。

### 3. 房间名称提示

进入新房间时，屏幕上方居中显示房间名，淡入→停留→淡出。

### 4. 暂停菜单

按ESC暂停游戏，全屏半透明遮罩 + "PAUSED"提示框。

| 类型 | 模型 | 行为 |
|------|------|------|
| 近战 (Melee) | robot4 | 冲上去打，攻击范围65px |
| 远程 (Ranged) | robot8 | 保持距离射击，攻击范围250px |
| 混合 (Hybrid) | robot6 | 远距离射击(200px)，近距离切换近战(80px) |

### 混合型敌人行为

```
距离 > 200px  → 靠近
80 ~ 200px    → 停下来远程射击
距离 < 80px   → 切换近战攻击
```

## 关键代码解析

### HUD分层渲染

```cpp
// 1. 渲染世界（跟随摄像机）
Mat4 pv = proj * camera->GetViewMatrix();
spriteRenderer->SetProjection(pv);
// ... 渲染地图、敌人、玩家 ...

// 2. 渲染HUD（不跟随摄像机，固定在屏幕上）
Mat4 hudProj = Mat4::Ortho(0, screenW, 0, screenH, -1, 1);
spriteRenderer->SetProjection(hudProj);
// ... 渲染HP条、MP条 ...

// 3. 伤害飘字（世界空间，跟随摄像机）
spriteRenderer->SetProjection(pv);
// ... 渲染飘字 ...
```

### 瞬移到敌人身后

```cpp
void TeleportBehindEnemy() {
    // 找最近的活着的敌人
    Enemy* nearest = FindNearest(400px以内);
    
    // 计算"身后"位置：敌人面朝右→瞬移到左边，反之亦然
    float behindOffset = nearest->faceRight ? -70 : 70;
    player.pos.x = nearest->pos.x + behindOffset;
    player.faceRight = (nearest->pos.x > player.pos.x); // 自动面向敌人
    
    player.mp -= 20;
    player.teleportCooldown = 3.0f;
}
```
